Problem Statement

Why this matters

  • LLMs are vulnerable to prompt injection, obfuscation, and multi-step jailbreaks.
  • Vendor/model-level filters can miss obfuscated or gradual attacks and provide limited auditability.
  • Enterprises need centralized policy enforcement, audit trails, and operator workflows.

Solution: SentinelSwarm

Swarm Agents (one per request)

  • Watcher: rule-assisted LLM sentinel — detects encoding, homoglyphs, injections
  • Classifier: Azure Content Safety score (severity)
  • Analyst: LLM-based intent & blast radius reasoning
  • Responder: Enforces policy (BLOCK / MONITOR / QUARANTINE) and logs incident

Key outcomes

  • Block or quarantine adversarial inputs before they reach downstream LLMs
  • Provide human-readable rationale & incident IDs for compliance
  • Centralized enforcement across multiple model providers
Designed to maximize AI Integration & Intelligence Design (25%) and System Architecture quality (25%).

Architecture & Engineering

High-level flow

Client → SentinelSwarm Gateway (FastAPI)
  ↳ Watcher (rule hints + LLM) → Classifier → Analyst → Responder → Decision
  ↳ Send to downstream LLM only if allowed
Logs → Audit DB / SIEM → Operator Dashboard

Engineering decisions

  • FastAPI + uvicorn for low-latency gateway
  • Rule-assisted hints (regex + encoding checks) prime LLM for higher recall
  • OIDC-based GitHub Actions → Azure Web App deployment for secure CI/CD
  • Modular agents for testability, metrics, and independent updates

AI Integration & Intelligence Design (25%)

How LLMs are used

  • Watcher & Analyst use Azure OpenAI (low temperature) for deterministic, explainable decisions
  • Classifier uses Azure Content Safety to provide a deterministic score used by policy
  • LLM outputs constrained via strict JSON schema & fallback when parsing fails

Defenses vs single-model filters

  • Rule hints detect base64, URL-encoding, unicode homoglyphs early
  • LLM reasoning catches semantic and cross-language attacks
  • Ensemble produces rationale for operator review and audit
Explainability and deterministic prompts improve reviewer confidence — include prompt templates in appendix.
